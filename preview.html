Tool Manager · macOS

Luca

A minimalistic, lightweight, decentralised tool manager for macOS to manage project-specific tool environments.

curl -fsSL https://luca.tools/install.sh | bash Copy
Features
📦

Version-specific Installs

Pin exact tool versions per project without global PATH pollution.

🛡️

Project Isolation

Each project runs its own tool versions, zero conflicts.

⚡

Zero Configuration

One command. No lockfiles, no globals, no setup tax.

🤝

Team Sharing

Commit a Lucafile. Everyone gets identical tools.

Quick Start
Step 1 — Install a tool
luca install realm/swiftlint@0.62.0
Step 2 — Use it
swiftlint --help
tuist build
